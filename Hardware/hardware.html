Hardware

Hardware and software

Hardware is the physical components of a computer. These are the things you can touch. Software’s are the programs that can be executed on the computer. On this page we will discuss the different types of devices and their role.

Internal Hardware

Internal hardware refers to the devices that are within the computer. These devices are attached to the motherboard and are connected using busses. A bus is a microscopic set of parallel wires that carries electrical signals between devices within the computer. The list of common internal hardware devices are:

  • CPU
  • RAM
  • ROM
  • Cache
  • Motherboard
  • SSD/Hard drive

External hardware components are components you can touch. This includes the mouse, monitor, keyboard and any other physical component you can directly touch.

CPU

The CPU is usually referred to as the brain of the computer. The CPU is composed of the ALU, registers, control units and instruction-decoder, and controls the flow of information between these components and the RAM during the fetch-execute cycle. Finally, the clock speed is the number of instructions executed per second.

Storage Devices

There are different storage devices within the computer each having a different purpose. What differentiates each storage device is its access speed and volatility. A storage device that is volatile loses its contents when power is turned off, whereas non-volatile devices don’t. Devices that provide very fast access speeds are typically used for cache and devices that are volatile, cheaper but provide relatively fast access speeds are used for the RAM. Non-volatile devices are used as secondary storage. We will now discuss each of these devices in detail.

RAM & Cache

RAM is a type of memory, where each cell can be accessed using a specified address. RAM needs to be accessed frequently by the CPU therefore it is positioned closer to the CPU to allow faster access speeds. There are two different types of RAM, DRAM and SRAM. DRAM is used to store data temporarily that the CPU will be using while the SRAM is used for cache memory. DRAM needs to be refreshed continuously while in operation whereas SRAM does not. This is because DRAM uses capacitors to store charge which has a natural tendency to discharge. SRAM, however stores data as transistors providing faster access speeds with the price of higher costs.

ROM

ROM is read-only memory and therefore cannot be written to. The contents of ROM are what the processor first reads when the computer is switched on. The basic operations of the ROM include a hardware check-up, transferring data from the secondary storage to the RAM and loading the OS.

SSD and Hard drive (HDD)

HDDs and SSDs are both non-volatile memory and are referred to as secondary storage. HDDs store data magnetically allowing it to store data after the power has turned off. HDDs being mechanical have slower access speeds compared to RAM, thus a different technology is used in place of RAM.

Supercomputers & Mainframes

Supercomputers are computers capable of performing fast computation on data. These computations cannot often be parallelised as the second operation depends upon the first. Therefore these operations are performed in a serial manner. Mainframes however, deal with a very different problem. Mainframes are used to process thousands of independent queries simultaneously that require less computation, and used as large servers in big organisations.
